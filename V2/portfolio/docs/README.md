# Documentation du portfolio

Ce dossier contient la documentation et les notes techniques du site.

Structure du projet :
- `public/` : pages HTML accessibles par le navigateur
- `src/` : sources CSS, JS et données
- `components/` : fragments HTML réutilisables
- `server/` : serveur Node.js optionnel

Voir `projects-notes.md` pour des détails techniques sur les projets listés.
