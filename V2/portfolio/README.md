# Portfolio (structure)

Arborescence fournie pour un site portfolio statique simple.

Démarrage rapide (optionnel) :
1. Installer les dépendances pour le serveur de test :

   npm init -y
   npm install express

2. Lancer le serveur (dans `portfolio/`) :

   node server/app.js

Navigate to http://localhost:3000

Remarques : cette structure est un point de départ ; adapte les liens et le pipeline de build selon tes besoins (ex : copier CSS/JS dans `public/` lors du build).
